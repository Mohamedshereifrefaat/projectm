أهلاً بكم في موقعنا

هذا الموقع هو مشروع بسيط يعرض مهاراتنا في:

  • كتابة أكواد HTML
  • تنسيق الصفحات بـ CSS
  • إنشاء الروابط والجداول

HTML هي لغة بسيطة تتكون من وسوم تُكتب داخل أقواس زاوية < >، ولكل وسم وظيفة محددة، مثل عرض عنوان أو إدراج صورة أو إنشاء رابط.

تخيل أن صفحة الويب مثل بيت، ولغة HTML هي الأساس والجدران التي تحدد شكله. بدونها، لن يكون هناك مكان لوضع الأثاث أو الديكور.

بدأت HTML في أواخر الثمانينيات، ومنذ ذلك الوقت تطورت كثيرًا لتدعم الصوت والفيديو والجداول والنماذج، مما جعلها أداة أساسية لكل مصمم ومطور مواقع.
